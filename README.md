# Штука для СДВГшников как в рилсах/тик-тока/шортсах и тд
Два разноцветных куба летают по полю.

Если куб находится в своём "домене", то он неуязвим. В нейтральном "домене" все кубы уязвимы

Иногда на поле появляются кружочки разных цветов, позволяющие менять "домены" поля, на один из цветов кубов или на нейтральный

Когда кубы сталкиваются уязвимый(е) куб(ы) уменьшаются в размере

Симуляция заканчивается, когда один из кубов исчезает
